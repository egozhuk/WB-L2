Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77, 78, 79]
```

Срезом массива получили "новый массив".

Что касается памяти?
a, b ссылаются на один и тот же участок памяти, об этом надо помнить

a - len = 5, cap = 5

b - len = 3, cap = 4
